# Отчёт

## Список лексем с их номерами
1. Имя (идентификатор).
2. Служебные: `while`, `if`, `else`, `input`, `output`, `array`, `int`.
3. Числа без знака.
4. Числа без знака, с десятичной точкой.
5. Отдельные языковые символы: `+`, `-`, `*`, `/`, `=`, `;`, `,`, `(`, `)`, `{`, `}`, `<`, `>`, `[`, `]`.
6. Концевой ограничитель — `$`.

## Семантические программы
`c[i]` — текущий входной символ; `i` — итератор; `name` — строка; `n` — число (`int`).
1. Начало лексемы-слова: `name = c[i];`.
2. Начало лексемы-числа: `n = c[i] - ‘0’;`.
3. Пропуск символа: `i++;`.
4. Считана лексема, состоящая из единственного символа.
5. Добавление символа в лексему.
6. Продолжение лексемы-слова: `name += c[i];`.
7. Распознано слово; проверка на ключевое и уже существующий идентификатор.
8. Распознано целое число.
9. Продолжение лексемы-числа: `n *= 10 + (c[i] - ‘0’);`.
10. Символ конца цепочки (`end-of-file`).
11. Операция: `i–;`.
12. Ошибка при обработке символов «`<`», «`>`».

## КС-грамматика языка, где лексемы суть терминалы
01. `P → int IP | array AP | Q`.
02. `Q → aH = S; Q | input(aH); Q | output(aH); Q | if (D) {Q} KQ | while (D) {Q}Q | λ`.
03. `I → aM`.
04. `M → , aM | ;`.
05. `A → a[k]N`.
06. `N → , a[k]N | ;`.
07. `H → [S] | λ`.
08. `K → else {Q} | λ`.
09. `D → S < S | S > S`.
10. `S → S + T | S - T | T`.
11. `T → T * F | T / F | F`.
12. `F → (S) | aH | k`.

### Нетерминалы
1. `P` — начало.
2. `Q` — операторы.
3. `I`, `M` — перечисление переменных.
4. `A`, `N` — перечисление массивов.
5. `H` — доступ к элементу массива.
6. `C`, `B`, `O`, `E`, `D`, `K` — условия.
7. `S`, `T`, `F` — арифметика.

## КС-грамматика языка, преобразованная в нестрогую форму Грейбах
01. `P → int IP | array AP | aH = S; Q | input(aH); Q | output(aH); Q | if (D) {Q} KQ | while (D) {Q}Q | λ`.
02. `Q → aH = S; Q | input(aH); Q | output(aH); Q | if (D) {Q} KQ | while (D) {Q}Q | λ`.
03. `I → aM`.
04. `M → , aM | ;`.
05. `A → a[k]N`.
06. `N → , a[k]N | ;`.
07. `H → [S] | λ`.
08. `V → * FV | / FV | λ`.
09. `U → + TU | - TU | λ`.
10. `B → < SZ | > SZ`.
11. `K → else {Q} | λ`.
12. `D → (S)VUB | aHVUB | kVUB`.
13. `S → (S)VU | aHVU | kVU`.
14. `T → (S)V | aHV | kV`.
15. `F → (S) | aH | k`.
16. `Z → λ`.

## Семантические действия для генерации ОПС
1. `k` — запись в ОПС операнда из входной цепочки, распознанного лексическим анализатором (константа).
2. `a` — запись в ОПС операнда из входной цепочки, распознанного лексическим анализатором (переменная).
3. `□` — пустое действие.
4. `<ц>` — номер исполняемой программы.

Переменная `i` — счётчик (количество) элементов, сгенерированных в ОПС, инкрементирующийся при генерации очередного её элемента; `m` — размер памяти, выделенной для массивов и переменных.

01. `P → □□□ | □□□ | a□□□=□ | □□a□□i□ | □□a□□o□ | □□□□4□7□□ | 5□□□4□6□`.
02. `Q → a□□□=□ | □□a□□i□ | □□a□□o□ | □□□□4□7□□ | 5□□□4□6□`.
03. `I → 1□`.
04. `M → □1□ | □`.
05. `A → 2□3□□`.
06. `N → □2□3□□ | □`.
07. `H → □□ind | □`.
08. `V → □□* | □□/`.
09. `U → □□+ | □□-`.
10. `B → □□< | □□>`.
11. `K → □8□7`.
12. `D → □□□□□□ | a□□□□ | k□□□`.
13. `S → □□□□□ | a□□□ | k□□`.
14. `T → □□□□ | a□□ | k□`.
15. `F → □□□ | a□ | k`.

## Список операций ОПС
1. Арифметические: `+`, `-`, `*`, `/`.
2. Присваивание — `=`.
3. Индексация — `ind`.
4. Ввод/вывод — `i`, `o`.
5. Безусловный и условный переходы по метке — `j`, `jf`.
6. Создание переменных и массивов (`int` требует один операнд — имя переменной; `array` же два — имя массива и его размер).
